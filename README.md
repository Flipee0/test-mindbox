# TODO app

Test task for mindbox❤️

https://flipee0.github.io/test-mindbox/
## Run project

```bash
npm i && npm start
```

## Tests

```bash
npm run test
```

## Features
* Watch todos
* Add todo
* Remove todo
* Change todo completeness
* Remove completed todos
* Reorder todos with dnd
* Search todos by text
* Filter todos by completeness
* Dark/light themes
* En/ru languages
